# qBootloader
qBootloader is a multi-platform bootloader for 32-bit Linux.

## Requirements
* x86 32-bit CPU
* Linux with non fragmented kernel file
* sudo hdparm --fibmap /boot/<kernel> should show one continues block of sectors with bytes_offset 0
